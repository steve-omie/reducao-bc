# Calculadora de Redução da Base de Cálculo Tributária

Esta página é uma **calculadora de redução da base de cálculo tributária**. Ela permite simular como a redução da base de cálculo afeta o valor do imposto e a carga tributária final, usando um valor fixo de operação de **R$ 1.000,00**.

---

## Funcionamento

O usuário informa:

- **Alíquota Original (%)**
- **Carga Tributária Final Desejada (%)**
- **Percentual de Redução da Base (%)**
- **Percentual da Base Aplicada (%)**

Os campos são interligados: ao alterar um, os outros são recalculados automaticamente para manter a consistência dos dados.

---

## Resultados Exibidos

- **Base de Cálculo Reduzida**
- **Valor do Imposto**
- **Carga Tributária Efetiva**

Há um botão para preencher os campos com um exemplo.

Os valores são formatados para **moeda brasileira** e **percentuais com até 4 casas decimais**.

---

> **Observação:**  
> Os cálculos são apenas didáticos e não têm validade jurídica.
